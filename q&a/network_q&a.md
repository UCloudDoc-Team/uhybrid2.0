# 网络类常见问题

Q1：外网的相关操作中，哪些是实时生效的，哪些是实际通过工单操作实现的，具体的SLA是多少？

目前实时生效的操作有：固定带宽调整限速大小

实际通过工单操作的有：

| 操作名称                     | 操作SLA |
| ---------------------------- | ------- |
| 带宽计费方式切换             | 2天     |
| 95峰值带宽，调整保底带宽大小 | 2天     |
| 创建外网出口                 | 4天     |
| 删除外网出口                 | 4天     |
| IP段移入外网出口             | 4天     |
| IP段移出外网出口             | 4天     |
